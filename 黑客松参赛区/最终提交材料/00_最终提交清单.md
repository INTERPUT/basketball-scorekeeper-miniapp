# 篮球技术台自动化 - 黑客松最终提交清单

## 1. 项目一句话介绍

篮球技术台自动化是一款面向基层篮球比赛的微信小程序，用四位房间码连接技术员和观众，通过自然语言录入比赛事件，自动完成比分、犯规、暂停、逐节比分、赛后签字和归档报告生成。

## 4. 对照评分维度的提交重点

### 痛点真实性与解决效果（35）

基层篮球赛常见问题是技术台人手少、记录标准不统一、观众无法实时了解比分、赛后纸质表格整理慢。小程序把这些问题收敛为一条闭环：建赛、录入、实时查看、比赛结束、签字、归档导出。

### Vibe Coding 创新与执行力（30）

项目不是简单页面原型，而是用 AI 辅助完成需求整理、领域模型、规则引擎、云函数、自然语言解析、真实云开发接线、测试和 debug。过程中通过状态文件持续接力，形成可验证的工程闭环。

### 原型可用性与用户体验（25）

首版已经能在微信开发者工具中跑完整比赛：技术员录入自然语言事件，观众用房间码进入只读页，双方签字后生成中文 PDF 报告、正式计分表 HTML 和结构化 JSON。

### 演示效果与完成度（10）

建议视频聚焦 3 分钟完整故事：痛点 -> 建赛 -> 观众加入 -> 自然语言录入 -> 比赛结束 -> 签字归档 -> 中文 PDF 报告。
