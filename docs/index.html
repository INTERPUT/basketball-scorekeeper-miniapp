Hackathon Project · WeChat Mini Program

让一名同学，也能撑起一整场篮球技术台

篮球技术台自动化把建赛、计分、规则提醒、观众实时查看、赛后签字和正式留档整合到一个手机小程序里，让学校班赛更快开打，也更容易复盘。

查看项目主页 查看代码

它解决的是每场班赛都会遇到的问题

纸质技术台慢、容易漏记，观众看不到稳定的实时信息，赛后还要重新汇总、签字、留档。这个项目把技术台变成一条清晰的数字化流程：记录发生的每个事件，实时生成比赛状态，最后自动形成可复核的归档。

低门槛

不要求专业技术台团队，一名技术员即可完成首版所需记录。

实时透明

观众通过四位房间码进入，只读查看同一份比赛数据。

赛后完整

双方队长签字后生成正式计分表、PDF 报告和结构化数据。

从开赛到归档，一条完整闭环

01创建比赛
02房间码加入
03文本/语音录入
04确认后入账
05双方队长签字
06生成留档

小程序体验预览

首版已经跑通

项目已完成微信小程序、云数据库、云函数、语音识别、签字归档和导出链路。比赛数据以事件流为基础，技术台、观众页和导出结果来自同一份快照。

  • 文本和语音输入统一进入人工确认链路。
  • 得分、犯规、暂停、节次和规则提醒可自动更新。
  • 观众匿名只读，技术员拥有唯一编辑权限。
  • 归档版本包含签字、正式计分表、PDF 和结构化数据。
